lys documentation
===============================

*lys* is a hybrid CUI/GUI analysis system for scientific data inspired by several softwares such as Matlab and Igor. The characteristics of *lys* is as following.

- Flexibile data analysis by Python CUI (numpy, scipy, dask) and GUI (PyQt, matplotlib, pyqtgraph).
- Interactive publication-quality data visualization based on matplotlib supported by the original GUI interface.
- Interactive analysis and visualization of multi-dimensional array by original GUI tool *MultiCut*.
- Automatic parallel computation by dask.
- Reproducible data analysis (Data processing can be exported as a file).
- Both CUI and GUI can be customized by users.
- Open source: Free to use. You can confirm what is done in lys and change it.
- Fully documented and well tested.

To use *lys*, go to :doc:`install` and :doc:`tutorials/tutorial`.
